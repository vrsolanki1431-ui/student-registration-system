📚 Student Registration System (Django)
🚀 About Project
This is a simple Student Registration System built using Django.
It allows users to register students, store their data, and manage records easily.
✨ Features
Student Registration Form
Store student details in database (SQLite)
View all registered students
Simple and clean UI
Django admin panel support
🛠️ Tech Stack
Python 🐍
Django 🌐
SQLite 🗄️
HTML, CSS 🎨
📂 Project Setup (Run Locally)
Bash
# Clone repository
git clone https://github.com/vrsolanki1431-ui/student-registration-system

# Go to project folder
cd student-register-system

# Install requirements
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start server
python manage.py runserver
🔑 Create Admin User
Bash
python manage.py createsuperuser
